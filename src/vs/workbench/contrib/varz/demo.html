Big Red Buttons - Accessibility First

Note: All VARZ buttons are designed to be large, high-contrast, and easily accessible for visually impaired users. They feature red backgrounds, large text, and comprehensive keyboard support.

Accessibility Features

  • High Contrast: Red buttons with white text for maximum visibility
  • Large Size: Minimum 120px width, 48px height for easy targeting
  • Keyboard Support: F1-F5 keys for quick access
  • Screen Reader: Comprehensive ARIA labels and descriptions
  • Focus Indicators: Clear focus outlines for keyboard navigation
  • Hover Effects: Visual feedback with smooth transitions
  • Reduced Motion: Respects user's motion preferences

Keyboard Shortcuts

  • F1: Voice Input
  • F2: Read Code
  • F3: Run Code
  • F4: Save File
  • F5: Clear Editor
  • Ctrl+V: Toggle VARZ Toolbar

Voice Commands

VARZ supports natural language voice commands:

  • "Run the code" ‚Üí Executes current file
  • "Save file" ‚Üí Saves current document
  • "Clear editor" ‚Üí Clears the code area
  • "Read code" ‚Üí Reads current code aloud
  • "Voice input" ‚Üí Activates speech recognition
